2. Relational Plot

2.1 relational plots and subplots

Sometimes we suspect that the relationship may be different within certain subgroups. In the last chapter, we started to look at subgroups by using the “hue” parameter to visualize each subgroup using a different color on the same plot.

Now we’ll try out a different method relplot() (relationship plot) to creat a separate plot per subgroup. We call the “relplot()” method and use the “kind” parameter to specify what kind of relational plot to use - scatter plot or line plot.

By setting “col” equal to “smoker”, we get a separate scatter plot groups by “smoker”, arranged horizontally in columns. You can use the “col_wrap” parameter to specify how many subplots you want per row. If you want to arrange these vertically in rows instead, you can use the “row” parameter instead of “col”. We can also change the order of the subplots by using the “col_order” and “row_order” parameters and giving it a list of ordered values.

2.2 Customizing scatter plots

Seaborn allows you to add more information to scatter plots by varying the size, the style, and the transparency of the points.

All of these options can be used in both the “scatterplot()” and “relplot()” functions, but we’ll continue to use “relplot()” since it’s more flexible and allows us to create subplots.

We want each point on the scatter plot to be sized based on the number of variable with larger number having bigger points on the plot. To do this, we’ll set the “size” parameter equal to the variable name from the dataset.

We can make it easier by using the “size” parameter in combination with the “hue” parameter. To do this, set “hue” equal to the variable name.

 1

sns.relplot(
    x="horsepower", 
    y="mpg", 
    data=mpg, 
    kind="scatter", 
    size="cylinders",
    hue="cylinders"
)

plt.show()

Notice that because “size” is a quantitative variable, Seaborn will automatically color the points the same color with different depth, instead of different colors per category value like we saw in previous plots.

We use “hue” to create different colored points based on groups. Setting “style” equal to allows us to better distinguish these subgroups by plotting points with a different point style in addition to a different color.

    y="mpg",
    data=mpg,
    hue="origin",
    style="origin",
    kind="scatter"
)

plt.show()

Setting the “alpha” parameter to a value between 0 and 1 will vary the transparency of the points in the plot, with 0 being completely transparent and 1 being completely non-transparent.

2.3 Line plots

In Seaborn, we have two types of relational plots: scatter plots and line plots. Line plots are the visualization of choice when we need to track the same thing over time.

3. Categorical Plot

3.1 Count plots and bar plots

Count plots and bar plots are two types of visualizations that Seaborn calls “categorical plots”.

We call catplot() method and use the “kind” parameter to specify what kind of categorical plot to use. “catplot()” offers the same flexibility that “relplot()” does, which means it will be easy to create subplots if we need to using the same “col” and “row” parameters.

1
2
3
4

sns.catplot(x="Internet usage", data=survey_data,
            kind="count", row="Age Category")

plt.show()

Use the “order” parameter which accepts a list can specific ordering of categories. This works for all types of categorical plots.

Bar plots look similar to count plots, but instead of the count of observations in each category, they show the mean of a quantitative variable among observations in each category. To create this bar plot, we use “catplot”. Specify the categorical variable on the x-axis, the quantitative variable on the y-axis, and set the “kind” parameter equal to “bar”.

Of course, you can also change the orientation of the bars in bar plots and count plots by switching the x and y parameters.

3.2 Box plot

A box plot shows the distribution of quantitative data. The colored box represents the 25th to 75th percentile, and the line in the middle of the box represents the median. The whiskers give a sense of the spread of the distribution, and the floating points represent outliers.

We can use the “catplot()” function to crate a box plot. We’ll put the categorical variable on the x-axis and the quantitative variable on the y-axis, and specify kind="box".

As a reminder, “catplot” allows you to change the order of the categories with a list by “order” parameter.

3.3 Point plots

Point plots show the mean of a quantitative variable for the observations in each category, plotted as a single point.

Both line plots and point plots show the mean of a quantitative variable and 95% confidence intervals for the mean. However, there is a key difference. Line plots are relational plots, so both the x- and y-axis are quantitative variables. In a point plot, one axis - usually the x-axis - is a categorical variable, making it a categorical plot.

Since Point plot is a categorical plot, we use “catplot” and set “kind” equal to “point”. To remove the lines connecting each point, set the “join” parameter equal to False. You can also add caps to the end of the confidence intervals, set the capsize parameter equal to the desired width of the caps. Finally, you can turn the confidence intervals off by setting the “ci” parameter equal to None.

4. Customizing Seaborn Plots

4.1 Changing plot style and color

Seaborn has five preset figure styles which change the background and axes of the plot. You can refer to them by name: “white” (default), “dark”, “whitegrid”, “darkgrid”, and “ticks”. To set one of these as the global style for all of your plots, use the “sns.set_style()” method.

You can change the color of the main elements of the plot with Seaborn’s “sns.set_palette()” method. Seaborn has many preset color palettes that you can refer to by name, or you can create your own custom palette with list.

Seaborn has a group of preset palettes called diverging palettes that are great to use if your visualization deals with a scale where the two ends of the scale are opposites and there is a neutral midpoint.

Another group of palettes are called sequential palettes. These are a single color (or two colors blended) moving from light to dark values. Sequential palettes are great for emphasizing a variable on a continuous scale.

You can also create your own custom palettes by passing in a list of color names.

Finally, you can change the scale of your plot by using the “sns.set_context()” method. The scale options from smallest to largest are “paper” (default), “notebook”, “talk”, and “poster”.

Recall that “relplot()” and “catplot()” both support making subplots. This means that they are creating FacetGrid objects. In contrast, single-type plot functions like “scatterplot()” and “countplot()” return a single AxesSubplot object.

A FacetGrid consists of one or more AxesSubplots, which is how it supports subplots.

To add a title to a FacetGrid object, first assign the plot to the variable “g”. And then, you can set the title using “g.fig.suptitle()”. To adjust the height of the title, you can use the “y” parameter whose default value is 1.

To add a title to an AxesSubplot object, call the method g.set_titles(). You can also use the y parameter here to adjust the height of the title. Notice that a FacetGrid consists of one or more AxesSubplots, thus you can set title on each of subplot in a plot.

Since g is a FacetGrid object, using “g.fig.suptitle()” will add a title to the figure as a whole. To alter the subplot titles, call “g.set_titles()” repeatly to set the titles for each AxesSubplot. If you want to use the variable name in the title, you can use {col_name} to reference the column value.
